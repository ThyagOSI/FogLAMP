Backend
=======

Requirements
-------------

* Python 3.5.2
* Ubuntu 16.0.4 or up
* Postgres 9.6.3


Installation
-------------

Check ``./build.sh -h`` in ``src/python/`` directory for quick setup and run.


Note:
^^^^^

if using pycharm, make sure

PyCharm > Project Interpreter > Add local ``src/python/venv/<host machine>/bin/python``
